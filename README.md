Clinic Queue Management System (CQMS)
Course: PROG103 - Principles of Structured Programming  
Institution: Limkokwing University of Creative Technology, Sierra Leone

System Overview
The Clinic Queue Management System is a terminal-based Python application developed to digitize patient registration and queue handling in
local clinics. It follows the Structured Programming model, utilizing modular functions and efficient data structures to solve real-world
community challenges.

It Key Features
Patient Registration: Captures name, ID, age, and medical condition.
Queue Visibility: Displays all registered patients in the order of arrival.
Service Module: Implements a First-In, First-Out (FIFO) logic to serve the next patient in line.

Sustainable Development Goals (SDG)
This system is aligned with SDG 3: Good Health and Well-being. By improving the administrative efficiency of healthcare facilities through
digital queue management, the system helps reduce patient wait times and organized record-keeping for better community health outcomes.

Technical Implementation
Language: Python 3.10
Data Structures: Lists and Dictionaries
Principles: Modularity (at least two functions), Iteration, and Decision Structures
